## What is StreamHouse?

StreamHouse is an open-source media platform combining the best features of popular streaming services with social functionality. Think **Plex meets Discord meets Netflix** - all in one seamless experience.

| Feature | Traditional Services | StreamHouse |
|---------|---------------------|-------------|
| **Watch Parties** | Limited or paid | ✅ Built-in & free |
| **Personal Library** | Separate app needed | ✅ Integrated |
| **Social Features** | Basic comments only | ✅ Groups & communities |
| **Content Discovery** | Platform-specific | ✅ Unified search |

---

## Planned Features (v1 / MVP)
Features planned for the initial release:

### Movie Streaming
- Stream movies and shows directly from our API  
- High-quality playback on desktop and mobile  
- Basic playback controls: play, pause, seek, volume  

### Unified Content Discovery
- **Single search** across your library and online sources  
- **Custom metadata** support for personal collections  

### Everywhere Access
- **Progressive Web App (PWA)** - available on desktop and mobile  
- **Mobile-responsive** design that works beautifully  
- **Offline capability** for downloaded content (desktop-only)  
- **Dark/light mode** with system preference detection
- **Cross-device sync** - start on TV, finish on phone

## Privacy & Control

- **Self-hostable**: keep data on your own servers  
- **No tracking**: viewing data is never sold  
- **Local-first architecture**  
- **Encrypted communications** for watch parties  
- **Free as in beer and speech** - no cost to use, fully open-source  

> ⚠️ **Content Responsibility**: StreamHouse integrates with public media APIs. Users may encounter pirated or copyrighted content. Users are responsible for complying with local copyright laws and regulations when accessing such content.
